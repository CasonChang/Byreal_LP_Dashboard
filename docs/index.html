注意事項

  • 年化採「資金×時間加權」計算。若部位中途加倉，年化會有些微誤差（後加的資金會被當成全程在場而略為低估）。若要投入大筆新資金，建議關單重開，以維持年化計算準確。
  • 「損益 / 總損益」僅反映代幣價格變動與無常損失（IL），不含手續費。
  • 超短持倉（如數十分鐘）的「年化」是把瞬間速率外推一整年，數字會偏大，僅供參考。
  • 調倉（改區間）在 CLMM 等於「關舊單、開新單」，因此每次調倉都會在歷史中新增一筆已關閉部位。
